## STARTING OFF:

Make sure you have npm installed.

Run:
```
$ npm install
$ npm start
```

View at: http://localhost:8080/

## CONCEPTS:

ImmutableJS
Record:
*  A record is similar to a JS object, but enforce a specific set of allowed string keys, and have default values.
