## STARTING OFF:

Make sure you have npm installed.

Run:
```
$ npm install
$ npm start
```

View at: http://localhost:8080/

## CONCEPTS:

ImmutableJS Iterator - Conversion to Collections, Conversion to Seq, Iterators
